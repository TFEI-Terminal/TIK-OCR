Please download all files located within this repository and save them to a designated folder on your local machine. 
Next, locate and open the 'demo.m' file within the folder, and run the code using Matlab version 2022b or above. 
Please note that the code has been tested and is expected to function correctly on the specified Matlab versions.


Please be advised that due to size restrictions on Github, files larger than 25MB cannot be hosted on the platform.
Therefore, if you require access to the trained Resnet18 files 'Net_resnet18v2.mat' and 'net_resnet18v4_error2.mat', please contact yxl875@psu.edu directly.
We apologize for any inconvenience this may cause.
